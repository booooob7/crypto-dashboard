# Demo 背景知識小抄 — 加密貨幣儀表板

> 用途：demo 時老師問「這個數字是什麼意思」可以對答如流。
> 每個名詞都附「一句話講法」與「被追問時的補充」。

---

## 0. 先講大架構（30 秒開場）

我的儀表板資料來自**三個面向、三個 API**：

| 面向 | 來源 | 回答什麼問題 |
|------|------|------------|
| **市場面** | CoinGecko | 現在幣值多少？市場多大？ |
| **情緒面** | Alternative.me | 投資人現在貪婪還是恐懼？ |
| **鏈上面** | Blockchain.com | 區塊鏈上實際有多少活動？ |

一句話：**「我從價格、情緒、鏈上三個角度，給市場一個全貌。」**

---

## 1. 價格相關名詞

### 價格（Price）
- **一句話**：這個幣現在值多少美元。
- 補充：我存的是「收盤價」快照，每 15 分鐘記錄一次。

### 市值（Market Cap）
- **一句話**：這個幣的「總價值」＝ 價格 × 流通數量。
- 補充：市值越大代表這個幣越主流、越難被少量資金撼動。我的儀表板抓「市值前十大」就是用這個排名。

### 24 小時成交量（Volume / 交易量）
- **一句話**：過去 24 小時，這個幣在**各大交易所**被買賣的總金額。
- 補充：成交量大 = 交易熱絡、流動性好。⚠️ 注意這跟「鏈上交易量」是**兩回事**（見第 4 點）。

### 24h / 7d 漲跌幅
- **一句話**：跟一天前 / 七天前比，漲了或跌了百分之幾。
- 卡片上綠色是漲、紅色是跌。

## 2. 幣種：穩定幣是什麼？

榜上會看到 **USDT、USDC**，它們是「穩定幣（Stablecoin）」：

### USDT（Tether）/ USDC（USD Coin）
- **一句話**：與美元 1:1 掛鉤的加密貨幣，1 顆 ≈ 1 美元，價格幾乎不動。
- **USDC** = USD Coin，由 Circle 公司發行；**USDT** = Tether，市場最大的穩定幣。
- 為什麼會在前十大？因為大家用它們當「避風港」和交易媒介，需求量巨大，所以市值很高。
- 被問「為什麼這兩個價格都是 1 塊、線是平的？」→ 因為它們設計上就是要錨定美元，不波動才是正常。

### 其他主流幣（順便記）
| 代號 | 名稱 | 一句話 |
|------|------|--------|
| BTC | Bitcoin 比特幣 | 第一個、市值最大的加密貨幣，「數位黃金」 |
| ETH | Ethereum 以太幣 | 智能合約平台龍頭，很多應用建在上面 |
| BNB | 幣安幣 | 全球最大交易所幣安的平台幣 |
| SOL | Solana | 主打高速、低手續費的公鏈 |
| XRP | Ripple | 主打跨境支付 |
| DOGE | 狗狗幣 | 迷因幣代表 |

## 3. 恐懼貪婪指數（Fear & Greed Index）

- **一句話**：一個 0~100 的分數，衡量整個市場現在的情緒。
- **怎麼讀**：
  - 0~25 = 極度恐懼（大家在恐慌賣出）
  - 25~45 = 恐懼
  - 45~55 = 中性
  - 55~75 = 貪婪
  - 75~100 = 極度貪婪（大家在瘋狂追高）
- **怎麼算的**：綜合波動率、市場動能、成交量、社群媒體聲量、市場主導率等指標。
- **投資含義**（被追問可講）：常被當「反指標」——「別人貪婪時恐懼，別人恐懼時貪婪」。極度恐懼有時反而是買點。

## 4. 鏈上數據（On-Chain Data）★ 這段最容易被問

### 什麼是「鏈上數據」？
- **一句話**：直接從區塊鏈帳本上讀到的真實活動紀錄，不是交易所的數字。
- 補充：區塊鏈是公開帳本，每一筆轉帳都記在上面、任何人可查。「鏈上數據」就是統計這些帳本紀錄，反映**網路被使用的真實程度**。
- 跟價格的差別：價格是「大家覺得它值多少」；鏈上數據是「實際上有多少人在用」。

### 活躍地址數（Active Addresses / n-unique-addresses）
- **一句話**：當天有實際參與交易的**不重複錢包地址**數量。
- 補充：可以想成「今天有多少個錢包動了」。地址數上升 = 使用者活躍、網路熱絡，常被當作網路健康度與採用度的指標。
- （口語「活躍地址」，不是「火藥地址」😄）

### 交易筆數（n-transactions）
- **一句話**：當天區塊鏈上確認了多少筆交易。
- 補充：反映網路的繁忙程度。筆數越多代表鏈上活動越多。

### 估計鏈上轉帳金額（Estimated Transaction Volume USD）
- **一句話**：當天在比特幣**區塊鏈上實際轉移**的價值，換算成美元。
- ⚠️ **最重要的區分**：這跟第 1 點的「24h 成交量」**完全不同**：
  - **交易所成交量** = 在 Binance 等平台上「買賣」的金額（換手）。
  - **鏈上轉帳金額** = 真的把幣從一個錢包「搬」到另一個錢包的金額（真實轉移）。
- 被問差別時就講：**「一個是交易所裡的買賣，一個是鏈上實際的轉帳，我兩個都有呈現。」**

## 5. 技術面可能被問（簡短答法）

- **「資料多久更新？」**→ GitHub Actions 排程設定每 15 分鐘自動抓一次，寫進資料庫。
- **「為什麼短期圖比較密、長期圖一天一點？」**→ 我做了自動解析度：7 天內顯示小時級日內、30/90 天收斂成日線，兼顧細節與易讀。
- **「資料存在哪？」**→ 雲端的 Supabase PostgreSQL，三張表分別存價格、情緒、鏈上。
- **「怎麼避免重複資料？」**→ 用資料庫唯一鍵 + upsert，重複跑也不會產生重複列（冪等）。
- **「為什麼用 Streamlit 不用 Tableau？」**→ 作業規定不能用 BI 工具，且我想自己掌控前後端與資料管線。
